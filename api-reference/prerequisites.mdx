# API Prerequisites

Before you can call any AQL AI Developer API, you’ll need two values from the **AQL platform**:

1. **Log in to the AQL platform**  
   → Go to **Account Settings → APIs**.

2. **Get your Bot ID**  
   → Copy the **Bot ID** shown on the APIs page.

3. **Generate an API Key**  
   → From the same page, click **Generate API Key**.  
   → Copy the generated **API Key**.

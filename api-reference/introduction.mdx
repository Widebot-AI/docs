
# AQL AI Developer APIs

AQL provides core APIs for building applications with natural language processing, document retrieval, and voice capabilities. These APIs are designed with Arabic language support as a primary feature.

1. **AQL Chat API**  
   → Text generation and conversational AI with image analysis support.
2. **RAG Answer API**  
   → Query documents you've uploaded and get answers with source citations.
3. **RAG Indexing API**  
   → Upload files or URLs to create a searchable knowledge base.
4. **RAG Knowledge CRUD API**  
   → Create, read, update, and delete knowledge sources in your knowledge base.
5. **Speech-to-Text API**  
   → Convert audio files to text with Arabic dialect recognition.
6. **Text-to-Speech API**  
   → Generate audio from text using Arabic and other language voices.
